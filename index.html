Apprentice

A thoroughly commented, highly simplified WordPress theme intended for those learning to theme WordPress for the first time

Download this project as a .zip file Download this project as a tar.gz file

Introduction

Apprentice is a WordPress theme designed to be friendly to people tinkering with WordPress theming for the first time. It aspires to be thoroughly commented and cleanly coded. Robustness and completeness sacrificed when necessary in favor of making the basics clear, easy to understand, and easy to tweak.

The theme includes jQuery and some basic styles to serve as a starting point for the intrepid student of WordPress.
